Place local fallback screen recordings in these folders. For production, prefer unlisted YouTube embeds by setting `VITE_HRMS_DEMO_YOUTUBE_URL` and `VITE_ERP_DEMO_YOUTUBE_URL`.

Recommended structure:
- `hrms/hrms-demo.mp4`: single HRMS walkthrough.
- `erp/erp-demo.mp4`: single ERP walkthrough.
- `crm/crm-demo.mp4`: lead automation, pipeline management, support/AI agent workflow.
- `document-intelligence/document-demo.mp4`: upload, OCR, retrieval, citation, audit trail walkthrough.
- `voice-bot/voice-bot-demo.mp4`: inbound/outbound voice bot and appointment/lead qualification demos.

Do not split HRMS into part 1 / part 2. Keep one demo per product.
